# About This Site

macOS desktop simulation built with [React](https://reactjs.org/), [Zustand](https://zustand-demo.pmnd.rs/), [UnoCSS](https://uno.antfu.me/), and [Vite](https://vitejs.dev/). Forked from [playground-macos](https://github.com/Renovamen/playground-macos) and rebuilt as my portfolio.

The terminal is fully interactive — try `help` to see what you can do. Bear has my resume split into sections. VS Code shows the actual markdown files.

Source code: [github.com/ud4yfr/CV](https://github.com/ud4yfr/CV)
