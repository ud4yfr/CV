# udayOS

Portfolio website simulating macOS, forked from [Renovamen/playground-macos](https://github.com/Renovamen/playground-macos).

Built with React + Zustand + UnoCSS + TypeScript + Vite.

## Features

- Interactive terminal with file system navigation
- Bear notes app with resume sections
- VS Code editor showing project files
- Safari with bookmarks
- FaceTime (webcam)
- Control Center with music player
- Dark/light mode toggle

## Setup

```bash
pnpm install
pnpm dev
pnpm build
```
